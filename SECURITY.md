# Security Policy

## Supported Versions

Handoff Relay is currently supported on the latest released version only.

| Version        | Supported |
| -------------- | --------- |
| Latest Release | ✅         |
| Older Releases | ❌         |

Security fixes will be applied to the latest version of the mod. Users are encouraged to update to the newest release before reporting a security issue.

## Reporting a Vulnerability

If you discover a security vulnerability in Handoff Relay, please report it privately and do not disclose it publicly until the issue has been investigated and resolved.

### How to Report

Please report vulnerabilities through one of the following methods:

* GitHub Issues (for non-sensitive reports)
* GitHub Security Advisories (preferred for security-related issues)
* Direct contact through Ko-fi or other listed project contact methods

### What to Include

When submitting a report, please include:

* A description of the vulnerability
* Steps to reproduce the issue
* Minecraft version
* Mod version
* Fabric Loader version
* Any relevant logs, screenshots, or crash reports

### Response Timeline

I will make reasonable efforts to:

* Acknowledge reports within 7 days
* Investigate and validate the issue
* Provide updates when available
* Release a fix as soon as practical if the issue is confirmed

### Responsible Disclosure

Please do not:

* Publicly disclose the vulnerability before a fix is available
* Use the vulnerability to harm servers, players, or data
* Attempt to access systems or data that you do not own

Valid reports that help improve the security and stability of the project are greatly appreciated.

Thank you for helping keep Handoff Relay secure.
